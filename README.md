# 👁️ Bogle's Eye (보글의 눈)

> "모든 주식을 소유하라. 그리고 비용은 최소화하라." - John Bogle

**Bogle's Eye**는 국내 상장 해외지수(S&P500, NASDAQ100) ETF의 마케팅 뒤에 숨겨진 
실질 비용(TER + 매매중개수수료)과 배당금, NAV를 투명하게 비교하여 
보글헤즈(Bogleheads)들이 상식적인 투자를 할 수 있도록 돕습니다.

## 🛠️ Key Features
- [ ] **실질 총보수 추적**: 기타비용과 매매수수료를 합산한 진짜 비용 공개
- [ ] **배당/분배금 현황**: 운용사별 분배금 지급 기록 비교
- [ ] **실시간 괴리율**: NAV 대비 현재가 거품 확인
